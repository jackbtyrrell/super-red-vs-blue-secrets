# Super Red vs Blue - Secret Items Guide

A guide to all known secret items and hidden features in **Typical Gamer's Super Red vs Blue** Fortnite Creative map.

**Map Code:** `2786-2114-8349`

## Secret Weapons (Item Wall)

| Weapon | How to Unlock |

| Treecapper | Found on the secret item wall |

## Secret Vaults

| Vault | Details |
|-------|---------|
| Secret Vault | Requires a passcode to enter — contains hidden treasures |
| Mystery Vault | Has its own unique vault code |
| Bomb Vault | Contains explosive items — helps you level up fast |

> Vault codes are frequently changed by the map creator. Check TikTok and YouTube Shorts for the latest codes.

## Secret Areas

- **Secret Room** — Accessed via a rare teleporter, contains hidden gems and collectibles
- **Secret Wall** — A hidden wall section that can be discovered in the map

## Other Features

- **Sheriff Role** — Become the sheriff and send other players to jail
- **Ranking System** — Rank up from Bronze to Unreal (progress saves between sessions)
- **Gold Coins** — Earn 10 Gold Coins per elimination
- **Achievements** — Complete in-game achievements for rewards
- **Super Powers** — Various super powers available in the map
- **Supply Drops** — Super Vault and Supply Drops available
